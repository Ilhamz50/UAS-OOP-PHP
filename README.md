# Sistem Absensi Siswa DKV

Aplikasi absensi berbasis PHP Native dengan konsep OOP.

## Fitur
- Login Admin & Siswa
- CRUD Data Siswa
- Update Status Absensi
- Role-Based Access

## Teknologi
- PHP Native
- MySQL
- CSS
- Laragon

## Cara Menjalankan
1. Import db_uas_dkv.sql
2. Simpan project di folder www/htdocs
3. Jalankan Apache & MySQL
4. Buka localhost/UAS-OOP-PHP
